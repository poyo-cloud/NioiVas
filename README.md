# 匂い研究 VAS アプリ

iPad 向けの静的 Web アプリです。コントロール 1 回と条件 1〜7 に対応する計 8 回の主観評価を実施し、端末内保存と CSV 出力を行います。

## できること

- 最初に `患者ID`、`検査実施日`、`パターン` を入力
- 同一ブラウザでは初回だけ 10cm 校正を要求
- 以後は必要な時だけ `校正` ボタンから再校正
- 校正は線を直接タッチまたはドラッグして調整
- 各評価回で 5 本の VAS と `reaction time` 3 回分を入力
- 被験者には `コントロール`、`1回目`〜`7回目` とだけ表示
- 端末内に履歴保存
- 単回または履歴まとめて `CSV` 出力

## 使い方

1. `index.html` を iPad の Safari で開く
2. `患者ID`、`検査実施日`、`パターン` を入力
3. 初回のみ、校正画面で線を実寸 10cm になるよう調整
4. `コントロール`、`1回目`〜`7回目` の順に 5 本の VAS と `reaction time` 3 回分を入力
5. `回答を終了する` で確定すると履歴に保存され、次の被験者入力へ戻る

## 保存について

- 下書き、履歴、校正値は `localStorage` に保存されます
- 校正値はこのブラウザに残るため、次回以降は基本的に再入力不要です
- 校正だけやり直したい場合は画面上部の `校正` ボタンを使います

## CSV 出力

- 1 人の被験者につき 1 行で出力します
- VAS アプリの CSV は `SubjectNo` `Pattern` に続いて VAS 列だけを出力します
- VAS 値は `B_Olf_threshold` から `C7_reactiontime` まで、条件番号順で並びます
- VAS 値は `0.0` 〜 `10.0` の cm 値で小数点第一位まで出力します
- `reaction time` は 3 回入力した値の平均を出力します
- 1 件ずつの出力ファイル名は `VAS_<患者ID>.csv` です
